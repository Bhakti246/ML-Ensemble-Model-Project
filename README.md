# 🔥 Ensemble Learning & Model Optimization Project

## 📌 Overview

This project demonstrates a complete Machine Learning pipeline using ensemble models like Random Forest and Gradient Boosting.

## 🚀 Key Features

* Logistic Regression (Baseline Model)
* Random Forest Classifier
* Gradient Boosting Classifier
* Model Comparison with Accuracy Graph
* Confusion Matrix Visualization
* Feature Importance Analysis

## 📊 Dataset

Breast Cancer Dataset (Scikit-learn)

## 🧠 Concepts Used

* Ensemble Learning
* Model Evaluation
* Feature Scaling
* Model Comparison

## 📈 Results

* Random Forest / Gradient Boosting achieved highest accuracy
* Visualization helped in better understanding of performance

## 🛠 Tools & Libraries

* Python
* Scikit-learn
* Pandas
* Matplotlib

## 💡 Conclusion

Ensemble models significantly improve prediction accuracy compared to basic models.
